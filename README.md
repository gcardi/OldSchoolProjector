## Key features

- **Full-screen projection** on the display of your choice, or in a window with a selectable aspect ratio (4:3, 16:9, 16:10).
- **Thumbnail strip** with asynchronous loading and caching: the current picture is highlighted and kept centered; click one to project it.
- **"Old school" ambiance**: the mechanical slide-change sound and the fan hum, with adjustable volumes and a sliding transition.
- **JPG & PNG support**, decoded through the Windows Imaging Component (WIC) for accurate colors.
- **Automatic EXIF rotation** — every image is turned the right way up from its EXIF orientation, so portrait shots are *never* sideways, in the thumbnails and in the projection alike.
- **"Contain" fit**: every photo is shown **in full** (letterboxed) with no cropping, portraits included.
- **Built-in test pattern** (monoscope) to calibrate and check the screen.
- **Image controls**: keep aspect ratio, clipping, scaling and a vintage **vignette** effect.
- **Pick any pictures folder**, with optional recursive search.
- **Network trigger** via a multicast event with an event code.
- **Tray icon** with minimize-to-tray and auto-start.

## Built with

C++ · Embarcadero C++Builder (FireMonkey / FMX) · Windows · Windows Imaging Component (WIC)
